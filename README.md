# AR Project

An Augmented Reality (AR) application built with Unity and Vuforia.

## Features

- Image Target-based AR experiences
- Real-time object tracking and recognition
- Interactive 3D content overlay

## Installation & Testing

### Option 1: APK File (Recommended for Quick Testing)

An Android APK file is available for immediate testing:

1. Download `ARapk.apk` from this repository
2. Transfer the file to your Android device
3. Install the APK (you may need to enable "Install from Unknown Sources" in your device settings)
4. Launch the app and point your camera at the image target to see AR content

### Option 2: Image Target Archive

For testing purposes, you can use the provided image target package:

1. Download `ImageTarget.rar` from this repository
2. Extract the archive to access the image target files
3. Print or display the image target on a screen
4. Use the AR app to scan the image target and trigger AR experiences

### Option 3: Build from Source

If you want to modify or build the project yourself:

1. Clone this repository:
   ```bash
   git clone https://github.com/alinorzadnuix-hub/AR.git
   ```
2. Open the project in Unity (2020.3 LTS or later recommended)
3. Ensure Vuforia Engine is installed via Package Manager
4. Build and run on your target platform

## Requirements

- **For APK Testing**: Android 7.0 (API level 24) or higher
- **For Development**: Unity 2020.3 LTS or later, Vuforia Engine

## Project Structure

- `Assets/` - Unity project assets including scripts, scenes, and resources
- `ProjectSettings/` - Unity project configuration files
- `Packages/` - Unity package dependencies
- `ARapk.apk` - Pre-built Android application
- `ImageTarget.rar` - Image target files for testing
